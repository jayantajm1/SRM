# WhatsFlow CRM

WhatsFlow CRM is a WhatsApp-first CRM for Indian small businesses. The current repository is being shaped into the MVP backbone for that product.

## Planning Docs

- [PRD](docs/whatsflow-crm/PRD.md)
- [Architecture](docs/whatsflow-crm/architecture.md)
- [MVP Scope](docs/whatsflow-crm/mvp-scope.md)

## Current Repo Layout

- `src/SRM.Api`: ASP.NET Core 9 API
- `src/SRM.Web`: Angular frontend
- `tests/SRM.Tests`: xUnit integration tests

## Current State

The repo currently contains a generic starter API and dashboard shell. The next iteration will rename and reshape those surfaces toward WhatsFlow CRM modules.
